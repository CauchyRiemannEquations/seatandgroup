# 🏫 스마트 클래스 자리배치 & 조 짜기 & 랜덤 뽑기

> **선생님을 위한 스마트하고 감각적인 북유럽 감성의 랜덤 자리배치, 조 짜기 및 랜덤 추첨 통합 플랫폼**  
> 별도의 설치나 가입, 서버 구축 없이 브라우저만으로 즉시 구동되며 오프라인에서도 완전한 로컬 데이터 보존 및 이미지 저장을 지원합니다.

---

## 🎨 주요 특징 (Features)

1. **감각적인 아카데믹 코지 테마 (Cozy Academic Theme)**
   - 차갑고 어두운 미래 지향형 네온 톤을 벗어던지고, 장시간 보아도 눈이 편안한 **북유럽 서재 감성의 오트밀 크림 라이트 테마**와 **Midnight 차콜 다크 테마**를 실시간으로 전환할 수 있습니다.
   - 빔프로젝터나 교실 TV 화면에서도 이름이 또렷하게 보이도록 글꼴 선명도와 가독성을 철저하게 극대화했습니다.

2. **유연한 교실 좌석 레이아웃 & 짝꿍 배치**
   - 5x6, 6x5 정형 그리드 배치뿐만 아니라 3분단 짝꿍 배치, 디귿(U)자 모둠형 배치를 원클릭 프리셋으로 로드할 수 있습니다.
   - 배치표 그리드 위에서 좌석을 직접 마우스로 클릭하여 간편하게 **통로(비활성 좌석)**와 **실제 좌석**을 자유로이 정의할 수 있습니다.

3. **기피 규칙이 상시 강제 적용되는 '조 짜기'**
   - 학생들을 일정한 명수로 묶어 균등하게 나누어 주는 스마트 조 짜기 도구입니다.
   - 등록된 학생 간 기피 쌍 규칙이 백그라운드 제약 해결기(Constraint Solver)에 의해 **상시 100% 강제 분리**되어 평화로운 모둠 구성을 보장합니다.

4. **🎰 아날로그 슬롯 롤링 랜덤 추첨 (Random Picker)**
   - 교탁 시계 틱(Tick) 효과음이 아날로그 슬롯머신처럼 흐르며 역동적으로 당첨 학생을 굴려 뽑습니다.
   - 추첨 성공 시 Stagger 딜레이로 차례로 날아와 뒤집히는 **황금빛 당첨 쿠폰(Golden Flip Ticket) 시각 연출**이 교실 내 집중도와 긴장감을 극대화합니다.

5. **💾 고해상도 이미지 다운로드 & 텍스트 복사**
   - 크롬/파이어폭스 등 현대적 브라우저 표준 보안 명세를 준수하는 캡처 모듈을 내장하여, 깔끔한 배치표를 고해상도 PNG 이미지 파일로 즉시 내려받거나 알림장에 공유할 텍스트로 즉각 복사할 수 있습니다.

## 🔒 비밀 관리자 모드 (Hidden Admin Mode)

학생들에게는 평범한 랜덤 배치 사이트로 보이지만, 선생님만의 은밀한 교실 관리를 돕는 **초강력 슈퍼 스텔스(Super-Stealth) 관리자 기능**을 제공합니다.

### 🗝️ 진입 방법 (Hidden Doors)
- **로고 5회 클릭**: 사이트 상단의 `Smart Classroom Seater & Grouper` 로고/제목 영역을 평온한 상태에서 **조용히 연속 5번** 클릭합니다. (마우스 커서가 포인터로 변하지 않아 아이들이 눌러볼 생각을 하지 못합니다.)
- **비밀 단축키**: 키보드로 `Ctrl + Shift + S`를 동시에 입력하면 무음 작동합니다.
- *※ 진입/해제 시 시선 집중을 유발하는 팝업창이나 알림 소리가 일절 나지 않으며, 오로지 왼쪽 하단에 숨어있던 `비밀 제약 설정` 창만 부드럽게 등장합니다.*

### 🛠️ 제공 기능
1. **은밀한 자리 고정 (Stealth Seating Lock)**
   - 비밀 모드가 켜진 상태에서 좌석 카드를 클릭하면 **인라인 이름 선택창**이 나타납니다. 여기서 지정된 학생은 해당 좌석에 즉시 묶입니다.
   - 중복 고정 방지 기능이 내장되어 있어 이미 다른 자리에 고정된 학생을 고르면 이전 고정은 알아서 해제됩니다.
   - **완벽한 알리바이 셔플**: 비밀 모드를 끄면 고정된 이름이 일반 **"좌 석"** 문구로 감쪽같이 감춰집니다. 또한 셔플을 돌릴 때 고정된 카드 또한 **다른 카드들과 똑같이 흔들리고 무작위 이름이 스쳐 지나가도록 롤링 셔플 연출**이 유기적으로 흘러가 RIGGED 흔적을 완전히 소거합니다.
   - **소스코드 스텔스**: 단순히 숨김 스타일링(`display: none`) 처리를 한 것이 아니라 HTML DOM 트리 구조에서 빨간 핀 요소를 아예 렌더링하지 않아, 똑똑한 학생들이 개발자 검사 도구를 열어도 고정 사실을 물리적으로 탐지할 수 없습니다.

2. **학생 기피 규칙 등록**
   - 절대로 짝이나 동일한 모둠 조에 묶여서는 안 되는 학생들을 등록하여 안전한 관계 평화를 유지합니다.

## 🌐 깃허브 Pages로 무료 배포 및 사용 방법

이 프로젝트는 별도의 데이터베이스나 서버 백엔드가 없는 무설치형 순수 HTML5 앱이므로, 깃허브에서 무료로 제공하는 호스팅 서비스인 **GitHub Pages**를 사용해 1분 만에 온라인 웹사이트로 배포할 수 있습니다.

### 🚀 1분 무료 배포 가이드
1. 깃허브에 새 저장소(New Repository)를 생성하고 `Public`으로 설정합니다.
2. 폴더 내의 핵심 파일들(`index.html`, `index.css`, `app.js`, `README.md`)을 그대로 저장소에 업로드(Commit)합니다.
3. 저장소 상단의 **Settings** 메뉴로 이동합니다.
4. 왼쪽 메뉴 바에서 **Pages** 탭을 클릭합니다.
5. Build and deployment의 Branch 설정을 `None`에서 `main` (혹은 `master`)으로 변경하고 옆의 **Save** 버튼을 누릅니다.
6. 약 30초~1분 뒤 상단에 자동으로 생성된 나만의 고유 사이트 링크(`https://계정이름.github.io/저장소이름/`)가 나타납니다.
7. 해당 주소로 접속하면 즉시 스마트폰, 태블릿, 교실 컴퓨터 어디서나 자유롭게 활용이 가능합니다!
